# StudyConnect

**Team:** Amil Ismail Khadida (315460)  
**Repository:** https://github.com/DEIN-REPO-LINK

## Project Idea
StudyConnect is a prototype web application that helps students find study partners.  
Users can search for partners based on subject, level, and learning format.

## URL Structure
The main page is accessible via `/`.  
Resources such as the stylesheet are loaded via `styles.css`.

## Criteria Mapping M1

| Criterion | File | Location |
|----------|------|---------|
| Semantic HTML structure | index.html | header, nav, main, section, article, footer |
| Form with labels | index.html | search section |
| Responsive layout (Flexbox/Grid) | styles.css | header (flex), grid-container (grid) |
| Media Query | styles.css | @media (max-width: 600px) |
| URL structure | index.html | links and resources |

## Note
This project is a static HTML/CSS prototype without JavaScript.
